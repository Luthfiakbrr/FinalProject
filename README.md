# Automation Test Framework

## 🚀 Teknologi
- Java + Gradle
- Cucumber (BDD)
- Selenium (UI Testing)
- RestAssured (API Testing)
- Allure (Reporting)
- GitHub Actions (CI/CD)

## 📁 Struktur
- `api/` → API tests (dummyapi.io)
- `web/` → UI tests (demoblaze.com)
- `hooks/` → Hooks for browser and reports
- `runners/` → Cucumber test runners

## ▶️ Menjalankan Test

```bash
./gradlew runApiTests
./gradlew runWebTests
